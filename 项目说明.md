# 人生故事记录AI (Apstory)

一款利用AI与用户对话，帮助用户记录、整理个人人生故事的智能应用。

## 项目状态

⚠️ **开发中** - 部分功能已实现，AI对话功能需进一步调试

## 功能特性

- 🤖 **AI对话引导** - 通过AI引导回忆和讲述人生故事
- 📝 **智能生成** - 自动整理对话内容生成结构化故事
- 🔒 **本地部署** - 数据完全存储在本地，保护隐私
- 🔧 **灵活配置** - 支持云服务API和本地大模型
- 📱 **响应式设计** - 适配各种设备

## 技术栈

### 前端
- React 18 + TypeScript
- Tailwind CSS
- Zustand (状态管理)
- React Router
- Vite

### 后端
- Node.js + Express
- SQLite (sql.js)
- DeepSeek API / OpenAI API

## 快速开始

### 环境要求

- Node.js 18+
- npm 9+

### 1. 安装依赖

```bash
# 安装后端依赖
cd backend
npm install --ignore-scripts

# 安装前端依赖
cd ../frontend
npm install --ignore-scripts
```

### 2. 配置

**后端配置** (`backend/.env`):
```env
PORT=3001
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=你的API密钥
DEEPSEEK_API_ENDPOINT=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-chat
DATABASE_PATH=./data/stories.db
```

**前端配置** (`frontend/.env`):
```env
VITE_API_BASE_URL=http://localhost:3001
VITE_APP_TITLE=人生故事记录AI
```

### 3. 启动服务

```bash
# 启动后端 (终端1)
cd backend
npm run dev

# 启动前端 (终端2)
cd frontend
npm run dev
```

### 4. 访问应用

- 前端: http://localhost:5173
- 后端API: http://localhost:3001/api

## 配置AI服务

### DeepSeek (当前使用)
```env
AI_PROVIDER=deepseek
DEEPSEEK_API_KEY=你的密钥
DEEPSEEK_API_ENDPOINT=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-chat
```

### OpenAI
```env
AI_PROVIDER=openai
OPENAI_API_KEY=你的密钥
OPENAI_API_ENDPOINT=https://api.openai.com/v1
OPENAI_MODEL=gpt-3.5-turbo
```

### Ollama (本地)
```env
AI_PROVIDER=ollama
OLLAMA_API_URL=http://localhost:11434
OLLAMA_MODEL=llama2
```

### LM Studio (本地)
```env
AI_PROVIDER=lmstudio
LMSTUDIO_API_URL=http://localhost:1234
LMSTUDIO_MODEL=你的模型
```
